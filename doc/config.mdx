# 🏗 配置

## 1. webpack

### 1.1 功能

- 代码切割
- less(配合 antd,如后期去除 antd，则删除此插件)
- babel (如有可能，还是希望使用 typescript 搭建，后期不断完善)
- antd,ant-mobile,emotion 的按需加载

### 1.2 环境

- html 文件自动生成及文件自动引用注入
- babel 编译 typescript 及向下兼容的包引用
- devServer 的路由配置及良好的热更新
- webpack 开发环境及生产环境区分

### 1.3 性能及体验

- 良好的控制台信息输出
- 对整个文件进行细粒度的打包分配
- 极快的打包速度，及生成运行环境
