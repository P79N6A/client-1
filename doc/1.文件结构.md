# 文件结构
```
  |---config                          ：webpack配置文件
  |   |--- webpack.common.ts          ：webpack公共配置项
  |   |--- webpack.dev.ts             ：webpack开发环境配置项
  |   |--- webpack.prod.ts            ：webpack生产环境配置项      
  |---doc                             ：文档
  |---public                          ：打包后的文件
  |   |--- index.html                 ：入口html文件
  |---src                             ：源代码  
  |   |--- components                 ：公共组件
  |   |--- layout                     ：页面结构
  |   |--- page                       ：页面组件
  |   |--- routes                     ：路由
  |   |--- store                      ：状态管理中心
  |   |--- index.tsx                  ：入口文件
```
