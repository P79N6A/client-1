# 商品 (cargo)

```
 //核心字段
- 商品名
- 商品编号
- 商品条码
- 物品图片
- 物品描述
- 状态（售卖中，下架，待上架）
- 分类
- 供应商
- 物品规格
- 物品单位
- 商品库存（根据物品规格划分）
- 零售价
- 成本价
- 生效日期
- 快递费（同意，包邮）
- 接入达达同城送

```
