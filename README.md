## 3. 📑 目录结构

文件新建请完全按照项目划分来创建

```
  |---config                          ：webpack配置文件
  |   |--- webpack.common.ts          ：webpack公共配置项
  |   |--- webpack.dev.ts             ：webpack开发环境配置项
  |   |--- webpack.prod.ts            ：webpack生产环境配置项
  |---doc                             ：文档
  |---public                          ：打包后的文件
  |   |--- index.html                 ：入口html文件
  |---src                             ：源代码
  |   |--- components                 ：公共组件
  |   |--- containers                 ：页面拆分组件
  |   |--- page                       ：页面组件
  |   |--- typing                     ：类型定义
  |   |--- redux                      ：状态管理中心
  |   |--- App.tsx                    ：根组件
  |   |--- index.tsx                  ：入口文件
```
